## Sobre

DateHappy é uma extensão de navegador desenvolvida como projeto final do curso CS50, criada para unir um desafio técnico real com um problema cotidiano: organizar datas importantes de forma simples, visual e intuitiva.

## Funcionalidades 

- **Adicionar eventos**: Clique diretamente sobre uma data no calendário ou use o formulário dedicado
- **Excluir eventos**: Remova eventos com um simples clique e confirmação
- **Mover eventos**: Arraste e solte eventos entre datas diferentes
- **Eventos com horário**: Utilize as visualizações `week` ou `day` para eventos com hora específica
- **Persistência de dados**: Todos os eventos são salvos localmente usando `chrome.storage.local`

### Eventos configurados
> Aniversário, Casamento, Natal, Ano Novo, Feriados, Carnaval, Páscoa, Halloween, Confraternização e Festa.

## Instalação
1. Clone ou baixe este repositório
2. Abra o Chrome e navegue até `chrome://extensions/`
3. Ative o "Modo do desenvolvedor"
4. Clique em "Carregar sem compactação"
5. Selecione a pasta do projeto
